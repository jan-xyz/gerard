# gerard

gerard is a slack bot written in Go.

## installation
```shell
go install ./gerard*.go
```
## usage
export your slack API KEY that you can get from your slack profile.
```shell
export ROLLMOPS_SLACK_API_KEY=<API-KEY>
go run ./gerard*.go
```
